# Marketing Analysis Dashboard

## Overview
Tableau dashboard analyzing product performance, 
channel efficiency, region-wise revenue and 
monthly conversion trends.

## Key Insights
- Product C leads all metrics (ROI 155.85, Revenue ₹3.45L)
- Direct Mail is the weakest channel (CPC ₹8.39)
- East region is most efficient (₹4.42 revenue per ₹1 spent)
- Q4 spend-conversion inversion identified

## Tools Used
- Tableau Desktop
- Excel (data source)

## Dashboard Sections
1. Product Performance
2. Channel Performance
3. Region & Product Revenue
4. Conversion V/S Spend + Final Summary
